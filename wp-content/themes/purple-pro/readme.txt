#Fonts Licence

Bundled font 'Dosis' used in the theme is GPL compatible.
Details about the font are here http://www.google.com/webfonts/specimen/Dosis

#How to install

1) Upload files to Themes directory (./wp-content/themes/).
2) Go to Dashboard - >  Appearance -> Themes and then choose the Purple Pro theme and click on Activate.

#Customization
Theme includes standard Wordpress features, custom header and footer menu and widgetized right sidebar.
Theme includes .po file and is translation ready.
